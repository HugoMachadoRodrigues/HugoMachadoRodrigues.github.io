Hugo Rodrigues

Postdoctoral Associate · University of Florida Pedometrics, Landscape Analysis & GIS Laboratory Gainesville, FL · Brazil

I am a soil scientist specialising in soil spectroscopy and pedometrics. My research applies statistical and deep learning methods — including supervised variational autoencoders and automated spectral resampling — to predict soil properties from Vis–NIR and SWIR spectral data. I emphasise methodological rigour in model calibration, open-source software development, and reproducible analytical pipelines.

Software & open-source packages
soilVAE R · Python / TensorFlow
Supervised variational autoencoder for regression with high-dimensional Vis–NIR–SWIR predictors. Exposes TensorFlow/Keras via reticulate. Targeting CRAN & SoftwareX.
GitHub Docs DOI
autoSpectra R · Shiny · v0.3.0
Soil spectral modelling, visualization and prediction using OSSL v1.2 and soilVAE. Covers VisNIR (350–2500 nm) and MIR (600–4000 cm⁻¹). Sensor-agnostic design.
GitHub Docs DOI
soilFlux R · CNN1D
Physics-informed CNN1D for soil water retention curves. Pedotransfer function model for continuous soil water content estimation across multiple matric potentials.
GitHub
soil-rf-spectral R · Random Forest
Soil Vis–NIR spectral modelling with tuned Random Forest on the OSSL/ASD pipeline. Reproducible calibration and independent validation workflow.
